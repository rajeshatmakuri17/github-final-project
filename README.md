markdown# Simple Interest Calculator

A simple, user-friendly tool designed to calculate the simple interest accrued on a principal amount over a specific period at a given interest rate.

## 🚀 Features

* **Quick Calculations**: Get instant results by entering basic financial figures.
* **Responsive Design**: Clean and fully responsive user interface (if applicable).
* **Validation**: Built-in input checks to prevent negative values or empty submissions.

## 🧮 Formula Used

The application calculates interest using the standard financial formula:

$$A = P (1 + rt)$$

Where:
* **$P$** = Principal amount (the initial sum of money)
* **$r$** = Annual interest rate (in decimal form)
* **$t$** = Time period (in years)
